## Token Lifetime

| Token                           | Default Lifetime |
| ------------------------------- | ---------------- |
| Access Token                    | 5 minutes        |
| Refresh Token                   | 4 hours          |
| Refresh Token (Offline Session) | <no expiry>      |
| ID Token                        | 5 minutes        |
