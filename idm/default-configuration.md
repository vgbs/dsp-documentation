## Token Lifetime

The following table lists the default lifetime configuration per token type. 

| Token                           | Default Lifetime |
| ------------------------------- | ---------------- |
| Access Token                    | 5 minutes        |
| Refresh Token                   | 4 hours          |
| Refresh Token (Offline Session) | No Expiry        |
| ID Token                        | 5 minutes        |
